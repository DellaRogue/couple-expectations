# 预期对齐（双人盲填）

静态网页：两人先后写下对同一事项的预期，提交后再一起揭晓，便于沟通时看见彼此、再协商平衡。

## 本地预览

直接用浏览器打开 `index.html`，或使用任意静态服务器（例如 `npx serve .`）。

## 部署到 GitHub Pages

1. 在 GitHub 新建仓库，把本目录中的文件推送到默认分支（例如 `main`）。
2. 打开仓库 **Settings → Pages**。
3. **Build and deployment** 里 Source 选 **Deploy from a branch**，Branch 选 `main`，文件夹选 **`/ (root)`**，保存。
4. 一两分钟后可通过 `https://<你的用户名>.github.io/<仓库名>/` 访问（具体 URL 以 Pages 提示为准）。

若首页 404，确认仓库根目录包含 `index.html`，且 Pages 指向的分支与路径正确。

## 隐私与限制

- 内容保存在浏览器 **sessionStorage**（关闭标签页即清空），不会上传到服务器。
- 适合 **同一台设备轮流填写**；若两人异地同时各用一台手机，当前版本 **无法自动同步**（需要额外后端或第三方实时数据库，若你需要我可以指导你在下次迭代里接入）。
